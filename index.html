Need Help?

We are here to help!

  • Join us on IRC at ##wicket on irc.freenode.net
  • Search our mailing list archives
  • Ask a question on the users list or at Stackoverflow.com
  • Suggest improvements for Apache Wicket on the developers list.
  • Report security related issues on the private list.

Need to Report a Bug?

Help us help you:

  1. Reproduce the bug with the least amount of code
  2. Create a unit test or a reproducer app that shows the bug
  3. Fix the bug and create a patch / Pull Request
  4. Attach the result of step 1, 2 or 3 to a bug report in:
    • Github tracker: for Wicket Stuff related bugs
    • Apache Wicket JIRA: for Apache Wicket related bugs
  5. Profit!
